    교재

    • 주 교재: 담당 교수와 NHN Academy가 제작한 강의 교재
    • 부 교재: Java의 정석 3판

    강의 계획

    1일차: Hello, World!

    즐거운 마음으로 자바의 세계로 오세요.

    강의 내용

    • 자바 프로그래밍 환경 준비
    • Hello, World! 출력
    • Java Progamming Language
    • 변수
    • 연산자

    실습 내용

    • 자바의 변수와 연산자를 활용할 수 있는 다양햔 실습을 진행해요
    • 자바와 관련된 리포트를 제출해요.

    2일차: 조건과 반복

    프로그래밍의 기본에 되는 각종 제어문을 배워요.

    강의 내용

    • 조건문
    • 반복문
    • 배열

    실습 내용

    • 강의 내용을 바탕으로 다양한 게임과 함수를 작성해요.

    3일차: 객체지향 프로그래밍 1

    객체지향의 기본을 자바를 통해서 배워요.

    강의 내용

    • 클래스와 객체
    • 속성과 행위 (변수와 메서드)
    • 오버로딩
    • 생성자
    • 초기화

    실습 내용

    • 스토리라인이 있으며 객체지향의 기본을 활용할 수 있는 실습을 진행해요. ex) *** 프로그램 작성

    4일차: 객체 지향 프로그래밍 2

    객체지향의 핵심을 자바를 통해서 배워요.

    강의 내용

    • 상속
    • 오버라이딩(재정의)
    • 패키지와 import
    • 제어자
    • 다형성
    • 추상클래스
    • 인터페이스
    • 중첩클래스

    실습 내용

    • 객체지향프로그램을 응용하는 실습을 전날에 이어서 진행해요.

    5일차: java의 기본 API와 예외 처리

    자바의 기본이 되는 api와 예외 처리를 배워요.

    강의 내용

    • Object, String,StringBuilder, Math, 레퍼클래스
    • Error과 Exception, 예외 계층구조, checked exception, try-catch, finally, try-with-resources

    - 실습 내용

    • 자바의 기본이 되는 유틸들과 예외처리를 다양한 실습을 통해서 익혀요.

    6일차: 더 많은 API를 배우자

    자바에서 많이 사용하는 api를 배워요.

    강의 내용

    • Objects, Random, java.util.regex, Scanner, StringTokenizer, BigInteger, BigDecimal
    • Calendar, Date, *Format, java.time

    실습 내용

    • 주로 수치와 수량 그리고 표현과 관련된 다양한 실습이 준비되어 있어요

    7일차: Collection Framework

    자바에서 다양한 자료구조를 표현하는 Collection Framework를 배워요.

    강의 내용

    • Collection
    • List
    • Set
    • Map
    • 대소 비교와 정렬

    실습 내용

    • 기존의 실습에 Collection Framework를 적용해요.
    • 그 외에도 자바에서 자료구조를 배울 수 있는 다양한 실습이 준비되어 있어요.

    8일차: 제네릭, 열거형, 애노테이션, 쓰레드(Threadlocal)

    java 1.5에서 추가된 새로운 기술들을 배워요. 그리고 쓰레드까지 알아보아요.

    강의 내용

    • 제네릭, 열거형, 애노테이션
    • 프로세스와 쓰레드, 쓰레드 구현과 제어
    • 멀티쓰레드
    • 쓰레드 동기화

    실습 내용

    • 이번 장에서 배운 내용을 기존 실습에 확장해서 적용해요.
    • 쓰레드와 동기화를 쉽게 익힐 수 있는 다양한 실습도 준비되어 있어요.

    9일차:람다와 스트림, I/O 1(표준 입출력과 파일)

    java 1.8에서 추가된 람다와 스트림을 알아보아요. 그리고 기본적인 I/O도 배워요.

    강의 내용

    • 함수형 인터페이스, ‘java.util.function’
    • 함수의 합성과 결합, 메소드 참조
    • stream(), map(), filter(). match(). collect(),
    • ‘Optional’

    실습 내용

    • 람다와 스트림을 통해서 함수형으로 구현하는 실습을 배워요.
    • I/O를 이용해서 프로그램에 불러오기/저장하기 기능을 구현해요.

    5일차: I/O 2, Networking

    자바로 네트워크를 다루는 법을 배워요.

    강의 내용

    • I/O: 바이트 기반 스트림, 문자열 기반 스트림, 직렬화
    • 네트워킹: InetAddress, URLConnection, 소켓 프로그래밍(TCP)

    실습 내용

    • 드디어 네트워크를 기반으로하는 채팅과 같은 실습을 할 수 있어요.
