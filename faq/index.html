경남 캠퍼스 2기 연수생 모집

웹 서비스 개발 과정에 응시하기 위해 갖추어야 할 지식은 무엇인가요?

  • 운영체제, 자료구조 등 기본적인 전산학 기초 지식
  • 간단한 수학적 문제를 프로그래밍 언어를 이용해 풀어낼 수 있는 능력
  • 정보기술 산업에서 사용되는 기술에 대한 이해

선발 시험은 어떻게 진행되나요?

  • 1차는 기술면접, 2차는 코딩 테스트로 진행됩니다.
  • 면접 전형료를 내신 분께 일정 및 안내 사항을 개별 메일로 전달해 드립니다.

교육 과정은 어떻게 진행되나요?

  • 풀타임(오전 9시-오후 6시) 오프라인 교육으로 진행됩니다.
  • 모든 교육은 오프라인으로 진행되며, Covid-19등 불가피한 경우에만 온라인 참석이 인정됩니다.
  • 교육 과정 구성은 아래와 같습니다.
    • 기초가 되는 Java 프로그래밍 언어와 웹 페이지 개발을 위한 기술을 배우는 기초 과정 3주
    • Java를 기반으로 동적인 응용 프로그램 개발할 수 있는 Spring Framework, 데이터를 저장하고 관리하는 데이터베이스, 네트워크, 응용 운영체제 유틸리티 등 개발에 도움을 주는 도구들을 이용해 프로그램을 작성하는 실무 과정 10주
    • 학습한 기술들을 사용해 기업형 응용 프로그램을 직접 작성하고 배포하는 프로젝트 과정 13주
  • 모든 교과는 이론 강의와 실습이 병행되어 진행됩니다.

수강료 결제는 어떻게 하나요?

  • 최종 선발된 연수생들에게 개별 메일로 결제 링크를 안내해 드립니다.
  • 카드 결제, 무통장 입금이 가능합니다.

최소 노트북 사양은 어떻게 되나요?

  • Windows의 경우 Windows 7 이상, CPU 인텔 코어 i3 이상, RAM 8GB 이상의 사양을 권장하며, macOS의 경우 Sierra(version 10.12) 이상의 사양을 권장합니다.

기숙사를 지원하나요?

  • 기숙사는 별도로 지원하지 않습니다.

코로나 확진 시 교육에 어떻게 참여하나요?

  • 코로나로 확진된 연수생에게 화상 회의 플랫폼을 통해 실시간 온라인 수업을 제공합니다.
