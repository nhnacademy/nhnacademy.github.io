P2.JavaScript

학습 목적

  • 웹 페이지를 브라우저에 표시하는데 사용되는 인터넷 프로토콜인 HTTP에 대해서 이해하고, 웹 응용 프로그램에서 사용자 브라우저에서 동작하는 기술인 HTML, CSS 및 JavaScript를 학습하여 웹 페이지를 제작하는 방법과 이를 구조화 하는 방법 및 사용자와 상호작용하는 웹 페이지를 제작하는 능력을 익힌다.
    • 웹 서비스와 HTTP 프로토콜의 이해
    • HTML 언어의 구조와 HTML5 태그를 사용하여 웹 페이지를 제작하는 방법 숙지
    • CSS3를 사용하여 반응형 웹페이지 등 계층화되고 구조화된 웹 페이지 제작
    • JavaScript를 사용하여 사용자와 상호작용하는 웹 페이지를 제작
    • 공개된 API를 구독하여 웹 페이지에 표시하는 방법 숙지
    • HTML Canvas를 이용하여 웹 페이지에 그림과 그래프 그리기

구성
강의 실습 과제

교재

  • 주교재: 담당 교수와 NHN Academy가 제작한 강의 교재
  • 부교재: HTML + CSS + 자바스크립트를 모두 설명할 수 있는 적당한 시중 서적

강의 계획

1일차: HTTP 개요 및 HTML 기본

강의 내용
  • HTTP의 역사와 동작 방식
  • 간단한 HTTP 테스트
  • HTTP Request와 Response
  • Header, Method, Status code
  • HTML의 역사와 HTML5
  • Element와 Attributes
  • HTML 문서 구조
  • HTML 메타 데이터
  • HTML 텍스트, 하이퍼 링크
실습 내용
  • 간단한 웹 서버를 만들어 실행하고 HTTP 테스트
  • HTML의 기초 태그들을 사용하여 정해진 포맷의 문서 생성

2일차: HTML과 CSS

강의 내용
  • HTML 레이아웃
  • HTML 구조화와 디버깅
  • CSS를 이용한 HTML 스타일 지정
  • CSS 박스 모델
실습 내용
  • CSS 스타일이 적용된 HTML 문서 작성
  • 동적 웹 페이지 작성

3일차: CSS, JavaScript

강의 내용
  • CSS 선택자
  • CSS를 이용한 트랜지션과 애니메이션
  • 반응형 웹과 미디어 쿼리
  • JavaScript의 역사
  • JavaScript 변수, 자료형, 연산자, 조건문, 반복문
  • JavaScript 함수
  • 함수 표현식
실습 내용
  • 반응형 웹 페이지 작성
  • JavaScript에서 Stack, Queue, Linked list, Tree, Map 사용

4일차: JavaScript

강의 내용
  • JavaScript 이벤트와 이벤트 핸들러
  • JavaScript 객체
  • JavaScript 내장 객체와 브라우저 지원 객체
  • Document Object Model
  • DOM을 이용한 이벤트 처리
  • 비동기 호출과 Proimse

실습 내용

  • DOM 모델을 이용한 사용자 상호작용적 페이지 생성
  • REST API 호출을 이용한 코로나 현황 조회

5일차: Canvas, Promise

강의 내용
  • ECMA 6에서 살펴볼 만한 것들
  • HTML5의 Canvas API 사용
  • 비동기 호출과 Promise
실습 내용
  • Canvas API를 사용한 그림 그리기

수행

  • Canvas에서 Bouncing Ball 구현
  • 비동기 호출과 Promise를 이용하여 코로나 현황 조회
