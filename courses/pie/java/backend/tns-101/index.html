1. 개발툴 익히기

학습 목적

  • 개발 환경 구성
    자바 통합 개발 환경을 사용하여 개발 환경을 구축하는 방법을 알고, 직접 실행할 수 있는 능력을 기르기.

    • IntelliJ IDE를 활용하여 Java 개발 환경 구성
    • 빌드 도구(Maven, Gradle)을 활용한 개발 환경 구성

  • Git & Github
    버전 관리 시스템에 대하여 이해하고, git을 사용해 프로젝트를 수행할 수 있는 능력을 기르기.

    • 버전 관리 시스템 개요
    • Git 기초 사용법 학습
    • 버전 관리 도구인 git-fork 활용하는 방법 학습

  • 빌드 도구
  • 빌드 도구 및 차이점
실습 내용
  • IntelliJ Learn IDE Features for Java 레슨

2일차: Gradle

강의 내용
  • Gradle 소개

  • Groovy
  • Gradle Task
  • Gradle 도메인 객체
  • 파일 관리
  • 플러그인

강의 내용
  • Maven 소개

  • Maven Wrapper

강의 내용
  • 버전 관리 시스템

  • 브랜치와 머지
