3. 개발에 도움을 주는 것들

학습 목적

  • vi editor의 사용법을 학습하고, shell script에 관한 기초지식 습득을 통해 리눅스 환경에서 시스템 자원을 이용한 도구 제작 기술을 익혀 보자.
  • 웹서비스 개발자가 알면 좋은 네트워크에 대한 기본 지식을 습득하자.
  • 서비스 서버 네트워크 상태 확인을 위한 여러가지 커맨드 학습하자.

교재

  • 담당 교수님과 NHN Academy 가 제작한 교제

강의 계획

1일차: vi 활용

강의 내용
  • vi 역사
  • 편집 명령
  • 중급 기법 - 검색과 치환
  • 고급 기법 - 외부와의 통신, 버퍼, 맞춤설정
  • vim 특수기능
실습 내용
  • Shell script 작성

2일차: Shell script 작성

강의 내용
  • Bash 소개
  • 커서 이동과 기본 명령
  • 변수
  • 파일 permission
  • 제어문
  • 입출력 리다이렉트
  • 필터
  • Alias
  • 함수
  • Script File
  • Job Control
  • 설정파일
  • 자주 사용하는 명령
실습 내용
  • 시스템 모니터링 스크립트 작성

3일차: 네트워크 구성 및 활용

강의 내용

  • 패킷 네트워크

  • Protocol Stack

    • OSI 7 layers
    • Protocols( Ethernet, IPv4, UDP, TCP, ICMP, DNS, HTTP)

  • 물리 구성

    • 라우터
    • 스위치
    • 허브

  • NAT

  • 로드밸런서

    • L4, L7
    • proxy (inline)
    • DSR

4일차: 유용한 Linux Utilities 활용

강의 내용

  • System 관리 툴

    • top
    • free
    • du

  • Network 관리 툴

    • ifconfig
    • netstat (ss)
    • ping
    • traceroute
    • nslookup (dig)
    • whois
    • tcpdump (tshark)
    • iptables
    • nc

  • 텍스트 처리 툴

    • cat
    • grep
    • awk
실습 내용
  • Network 관리 툴을 이용한 상태 확인

5일차: TShark

  • 강의 내용
  • 실습 내용

수행

  • Linux Shell 환경에서 정보 수집
  • 시스템 모니터링 보고서 작성
  • 패킷 캡쳐 후 분석 보고서 작성
