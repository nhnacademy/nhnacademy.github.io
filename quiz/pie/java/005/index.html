class F {
    X m() { System.out.println("F"); return new X(); }
}

class G extends F {
    Y m() { System.out.println("G"); return new Y(); }

    public static void test() {
        F f = new G();
        f.m();
    }
}

G.test();

G
  1. G가 찍히도록 X와 Y를 정의하세요.
  2. 이와 같은 method overriding 규칙으로 얻게 되는 이점은 무엇인가요?

도움말

귀띔

  • Covariant (method) return type 규칙을 되짚어 봅니다.

참고 문헌

  1. Covariant return type
  2. Liskov Substitution Principle(LSP)
  3. Covariance and contravariance
