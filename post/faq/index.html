Java 백엔드 개발자 연수생 모집

2022년 04월 21일 2 min read
Java 백엔드 개발 과정에 응시하기 위해 갖추어야 할 지식은 무엇인가요?

  • 운영체제, 자료구조 같은 전산학 기초
  • Python, Java로 기초 알고리즘을 프로그램하기

선발 시험은 어떻게 진행되나요?

  • 1차는 기술면접, 2차는 코딩 테스트로 진행됩니다.
  • 출제 범위는 NHN Track을 참고하세요.
  • 선발 시험 문제는 퀴즈에 공개합니다.
  • 면접 전형료를 내신 분께 일정 및 안내 사항을 개별 메일로 전달해 드립니다.

교육 과정은 어떻게 진행하나요?

  • 모든 교육은 풀타임(오전 9시-오후 6시) 오프라인으로 진행합니다.
  • Covid-19 감염과 같은 불가피한 경우에만 온라인 참석을 인정합니다.
  • 교육 과정은 아래와 같습니다.
    • Java 프로그래밍 언어와 웹 페이지 개발 기술을 배우는 3주 과정
    • Java 응용 프로그램 개발에 필요한 Spring Framework, 데이터베이스, 네트워크, 운영체제 같은 도구와 기술을 익히는 실무 10주 과정
    • NHN Dooray! 자바 백엔드 개발 경험을 전수받는 프로젝트 13주 과정
  • 모든 교육 과정은 이론 강의와 실습을 병행합니다.

수강료 결제는 어떻게 하나요?

  • 최종 선발된 연수생들에게 개별 메일로 결제 링크를 안내해 드립니다.
  • 카드 결제, 무통장 입금이 가능합니다.

최소 노트북 사양은 어떻게 되나요?

  • IntelliJ IDEA를 쓸 수 있는 사양이면 됩니다.

기숙사를 지원하나요?

  • 지역에서 일할 인재를 양성하기 위하여 특별히 마련된 프로그램이기 때문에 주거 시설을 별도로 지원하지는 않습니다.

Covid-19에 감염되면 교육에 어떻게 참여하나요?

  • 확진된 연수생은 화상 회의 플랫폼으로 실시간 온라인 수업에 참여할 수 있습니다.

※ 모집 설명회에서 교육 과정, 평가, 채용 연계에 대하여 상세하게 안내해 드립니다.
